About this Site

beforeitwasround is hosted on Github Pages and is a static site generated using Jekyll along with several plugins jekyll_alias_generator. Stylesheets are generated using Compass and Sass. Some javascript is generated using Coffee-Script. The logo is set in The League of Moveable Type's Goudy Bookletter 1911. The responsive layout is courtesy of Zurb's Foundation framework. Social Media icons are courtesy of VeoDesign's Somicro Icon Set and Picons Social. Tweets are loaded using Seaofcloud's Tweet jQuery plugin as well as Holman's Boastful plugin.
