About this Site

beforeitwasround is hosted on Github Pages and is a static site generated using Jekyll along with several plugins jekyll_alias_generator. Stylesheets are generated using Compass and Sass. Some JavaScript is generated using Coffee-Script. Tweets are loaded using Seaofcloud's Tweet jQuery plugin as well as Holman's Boastful plugin.

The site's design is my own, with some help (ok, a lot of help) from Twitter Bootstrap and Dan Cederholm's Pears pattern library. The color palette is Soaked. The logo is set in The League of Moveable Type's Goudy Bookletter 1911. Social media icons are courtesy of Socialico Icon Font.
