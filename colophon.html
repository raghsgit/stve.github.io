beforeitwasround is a static site generated using Jekyll. It is hosted on Github Pages. Stylesheets are generated using Compass and Sass. Some JavaScript is generated using Coffee-Script. Tweets are loaded using Seaofcloud's Tweet jQuery plugin as well as Zack Holman's Boastful plugin.

The site's design is my own, with some help (ok, a lot of help) from Twitter Bootstrap and Dan Cederholm's Pears pattern library. The color palette is one designed called Soaked. The logo is set in The League of Moveable Type's Goudy Bookletter 1911. Social media icons are courtesy of Socialico Icon Font.

So why did it take me so long to relaunch it? Truth be told, I worked on this site off and on over the span of several months. I used this as an opportunity to learn about Compass and Sass, dabble in CoffeeScript, as well as migrate the whole thing off of a self-hosted WordPress installation. I also really wanted the site to look good both visually and in the code. Along the way, I worked on guard-jekyll and fiddled with all sorts of things, including: Foundation, h5bp, among others.
