Colophon.

beforeitwasround is a static site generated using Jekyll and is hosted on Github Pages. Stylesheets are generated with the aid of Compass and Sass. 3rd-party libraries are managed using Bower. I use the awesome build system gulp.js to package everything together.

The site's design is my own, with some help (ok, a lot of help) from Coby Chapple's Gridism and Dan Cederholm's Pears pattern library. The color palette is one I designed called Soaked. The logo is set in The League of Moveable Type's Goudy Bookletter 1911. Social media icons are courtesy of Symbolset Social. Additional text is set in Calluna Regular

All source code for the site, along with the gory commit history can be viewed on github.
